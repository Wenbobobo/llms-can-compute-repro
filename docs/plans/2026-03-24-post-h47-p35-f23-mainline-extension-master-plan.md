## Why This Route

The origin note and the saved discussion files support a narrower scientific
reading than the headline:

1. the strongest current substrate evidence is still append-only exact
   execution plus bounded useful-case support;
2. `R48` is positive but explicitly comparator-only and non-substitutive; and
3. the next meaningful falsifier is not “can we build bigger demos,” but “can
   the landed exact useful-case surface survive one more narrow numeric and
   length extension without hidden scope lift.”

If that numeric extension fails under modest widening, the near-term
useful-case `wasm/tiny-C` mainline is practically falsified and should stop.

## Stop Rules

Stop the useful-case mainline early if either of the following occurs:

- `R49` needs a new substrate, hidden side channel, or undeclared scope lift;
- `R49` shows the landed exact stack does not survive modest numeric/length
  widening on the preserved useful-case surface.

Either outcome is strong enough to count as a practical falsifier of the
near-term “cleanly extend the current exact useful-case bridge toward tiny-C”
story.

4. push each green wave before starting the next one; and
