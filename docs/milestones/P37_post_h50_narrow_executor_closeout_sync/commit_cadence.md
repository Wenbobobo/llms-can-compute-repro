- commit `F28` planning surfaces separately from all packet or runtime work;
- commit `H51` docs-only control surfaces separately from `P37` hygiene
  surfaces;
- commit `P37` hygiene surfaces separately from scientific claims;
- commit `R55` exact retrieval-equivalence outputs separately from `R56`
  trace-semantics outputs;
