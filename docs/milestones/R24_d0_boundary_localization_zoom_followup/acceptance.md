# Acceptance

- one bounded candidate core is written down rather than leaving “hardest rows”
  abstract;
- one executable future matrix is written down and tied to the current `R21/R22`
  maxima it is meant to exceed;

  than curiosity alone;

  claimed localization;
- verdict vocabulary is explicit before execution;
- no run is started from this planning lane.
