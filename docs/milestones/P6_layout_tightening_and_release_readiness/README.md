# P6 Layout Tightening and Release Readiness

Goal: move from the completed `P5` sentence-polished manuscript bundle to a
layout-disciplined, release-ready paper package without reopening scientific
scope.

Scope:

- tighten figure/table placement and local callout flow inside the manuscript;
- keep caption sources, narrative roles, and section ownership aligned;
- keep README/STATUS and future short updates downstream of the manuscript via
  `release_summary_draft.md`;
- record layout-only decisions and handoff notes for later unattended waves.

Non-goals:

- no new experiments by default;
- no reopening of `R1`, `R2`, or `D0` scope without a separate evidence wave;
- no frontend widening;
- no blog release ahead of the paper-facing bundle.
