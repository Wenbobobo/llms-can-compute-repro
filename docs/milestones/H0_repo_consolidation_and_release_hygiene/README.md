# H0 Repo Consolidation and Release Hygiene

Goal: make the public surface honest, reproducible, and reviewable before any
future outward sync or scope reopening.

Scope:

- snapshot the current dirty worktree in machine-readable form;
- record a coherent preferred commit split instead of leaving the tree as one
  opaque batch;
- verify that root docs, ignore rules, and the current `M7` / `P4` gate
  outputs agree on the repo's restrained public posture.

Non-goals:

- no new scientific claim layers;
- no new demos or widened frontends;
- no attempt to rewrite the research direction after `M7` and `P4` have
  already resolved the current scope.
