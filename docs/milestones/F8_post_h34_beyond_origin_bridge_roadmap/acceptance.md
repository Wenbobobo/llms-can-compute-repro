# Acceptance

- `F8` remains planning-only and does not create a runtime authorization;
- the bundle identifies the current narrow Origin-core line as the only
  `supported_here` bridge surface;
- future families are classified explicitly as `planning_only`,
  `blocked_by_scope`, or `requires_new_substrate`;
- no document in this bundle implies that Wasm-like widening, richer compiled
  semantics, or hybrid planner-executor work is currently active;
- any later activation remains downstream of a future explicit packet rather
  than momentum from `R39` or `P20`.
