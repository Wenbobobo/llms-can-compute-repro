# F8 Post-H34 Beyond-Origin Bridge Roadmap

Planning-only roadmap downstream of `H34`, `P20`, `F5`, and `F6`.

`F8` exists to map what "beyond the current Origin-core line" could mean
without relabeling any future family as active science. It separates:

- what the repo already supports on the current Origin-core substrate;
- what remains planning-only as a future bridge surface;
- what stays `blocked_by_scope`;
- what already `requires_new_substrate`.

`F8` does not authorize runtime execution, compiled-family widening, broader
Wasm-like claims, or hybrid planner-executor work by itself.

Current conclusion:

- `current_origin_core_line_supported_narrowly`
- `beyond_origin_bridge_families_indexed_planning_only`
