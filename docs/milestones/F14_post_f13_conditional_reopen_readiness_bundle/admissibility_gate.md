# Admissibility Gate

A future contradiction-led reopen is discussable only if one candidate satisfies
all of the following at once:

1. same admitted row and same named boundary probe;
2. same opcode surface;
3. same substrate;
4. one named contradiction tied to either runtime irrelevance or easy-part-only
   acceleration;
5. one fixed comparator set already implied by `F13`;
6. one predeclared measurement rule;
7. one explicit stop condition;
8. no need for richer value families outside `bounded scalar locals and flags`.

If any condition is missing:

- keep the candidate at `inactive` or `still_nonunique`;
- do not promote it into a future packet blueprint.
