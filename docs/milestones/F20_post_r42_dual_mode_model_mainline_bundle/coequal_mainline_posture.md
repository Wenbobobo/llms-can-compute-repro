# Coequal Mainline Posture

`Coequal Mainline` means the project now carries one exact lane and one model
lane as first-class implementation programs after `R42`, but not as identical
evidence lanes.

The exact lane remains:

- `R43_origin_bounded_memory_small_vm_execution_gate`

The coequal model lane remains:

- `R45_origin_dual_mode_model_mainline_gate`

The exact lane is still decisive for the narrow Origin-core scientific claim.
The model lane is allowed to become operationally useful, diagnostically rich,
or even stronger on some bounded cases, but it does not replace exact evidence
for the main scientific statement.
