# Exact-Model Evidence Boundary

The evidence rule after `F20` is:

- exact `R43` evidence can directly support the bounded-memory small-VM claim;
- model `R45` evidence can support comparator, robustness, or operator
  usefulness claims on the same bounded kernels;
- model-only positives cannot stand in for exact `R43`;
- model-only failures do not invalidate a positive exact `R43`.

This boundary keeps the repo on the narrow Origin-core thesis while still
allowing a serious model program.
