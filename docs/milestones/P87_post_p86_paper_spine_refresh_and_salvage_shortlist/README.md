# P87 Post-P86 Paper Spine Refresh And Salvage Shortlist

`P87_post_p86_paper_spine_refresh_and_salvage_shortlist` refreshes the
paper-facing spine after the `P86` dirty-root inventory. It does not import the
dirty root wholesale. Instead, it updates the clean-branch paper summary docs
to the current `H65 + P86 + P85` posture and records which dirty-root
publication files are worth selective salvage next.
