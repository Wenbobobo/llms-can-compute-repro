# H25 Refreeze After R30 R31 Decision Packet

`H25` freezes the post-`H23` reauthorization wave.

This packet does not replace `H23` as the frozen scientific evidence state.
Instead it freezes the operational decision about which future lane, if any, is
still justified downstream of `H23`.

`H25` should therefore:

- preserve `H23` as the current frozen same-endpoint scientific state;
- preserve `R29` and `F3` as blocked future lanes;
- authorize at most one primary next science lane and one deferred audit lane;
- keep claim partition and non-goals explicit.
