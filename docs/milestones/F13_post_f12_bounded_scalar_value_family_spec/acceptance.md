# Acceptance

- `family_scope.md` fixes one bounded family and rejects nearby broader ones.
- `reference_semantics.md` and `projection_and_observability.md` together make
  the family explicit at source, trace, and final-state level.
- `lowering_invariants.md` and `comparator_matrix.md` define the minimum rules
  before a later packet could even discuss activation.
- `boundary_probe.md` plus `success_failure_rule.md` and `stop_conditions.md`
  make later evaluation bounded rather than exploratory.
