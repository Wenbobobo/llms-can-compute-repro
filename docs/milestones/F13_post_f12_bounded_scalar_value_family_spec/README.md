# F13 Post-F12 Bounded Scalar Value Family Spec

Planning-only preactivation bundle for the nearest richer same-substrate value
family identified by `F10`.

`F13` exists to turn `bounded scalar locals and flags` from a class-level row
in the `F10` value-family matrix into one decision-complete bounded spec. It
does not authorize runtime execution, restricted-Wasm work, hybrid work, or a
scope packet.

This bundle preserves:

- `H32` as the active routing packet;
- `H34` as the current docs-only control packet;
- `F10` as the current semantic/value bridge;
- `F9` as blocked and `F11` as new-substrate;
- no active downstream runtime lane.
