# Stop Rules

- stop if the proposed route collapses back into renamed same-lane executor
  value work;
- stop if exactness is not preserved fully on the declared route;
- stop if the proposal only beats Python or other non-honest comparators;
- stop if the route wins only on synthetic workloads and not on the lifted
  useful-kernel target;
- stop after one profiling pass plus one optimization pass;
- stop if the route fails to specify a materially different cost model; and
- stop if the route needs broader Wasm or arbitrary `C` widening to look good.
