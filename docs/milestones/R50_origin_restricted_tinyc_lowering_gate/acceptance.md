# R50 Acceptance

`R50` passes as an executed gate only if:

- all admitted rows stay inside the `F25` restricted tiny-`C` surface;
- exactness rows exist for all `8` planned cases;
- `3/3` kernel coverage and `8/8` variant coverage remain explicit;
- tiny-`C` lowering remains instruction-identical to the canonical bytecode
  kernels on all admitted rows;
- comparator/model evidence is not used as a substitute for exact
  `R47/R49` evidence; and
- the wave stops at `H49` rather than widening by momentum.
